Help

What is it ?

Match Maker Helping Tool

Match Up Tool is a website that will allow Matchmakers to handle most of their job. After creating a free account, it will be possible to add new candidates to their personals lists and find the best match for everyone.

Two mains features will make this possible. With the Messaging and the MatchFind features, matching peoples has never been that easy and comfortable !

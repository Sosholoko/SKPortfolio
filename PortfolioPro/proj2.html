What is it ?

Instant Messaging

Hover Messaging is WebMessaging Website that allows users connected to a common server to discuss by instant messages. Creating a new account is totally free.

It mainly uses Pusher solution to be fully functional. Pusher offers Bi-directional hosted API, it helps messages transiting though users.

Techs Used : HTML / CSS / Python / Javascript / Flask / SQLACHEMY / Sqlite3 / Pusher

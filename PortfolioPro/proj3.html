To Do List

Simple To Do List

What does it do ?

Add

Allow user to add new tasks that he planned to do ! The Plus Button will add the task to the ToDo list !

Reminds

The app has two separates lists, one if for the uncompleted items. It will help reminds the user to actually complete the task.

Complete

Once you finish complete the task, the "V" button will move the said task to the second list of completed items. Congratulation you did it !

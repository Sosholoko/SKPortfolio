Familygram

Instagram Clone for Family

What does it do ?

Simplicity

Each new users who want to join will be able to create his own Account and start using it directly ! The UI was thought to be as much intuitive as possible. Even for the olders one of us !

Posts

As soon as the user is connected, he'll be introduced to the Post Feed where all his connections already have posted pictures, he'll be able to also post anything just by touching the Plus Button, simple right ?

Interact

For any kind of social networks wanting to be considered legit, it'll need the basics interactions such as Like, Comment and Share, Familygram is in the game too ! Like Instagram those 3 options will be displayed at the bottom of each posts.

What is it ?

Fully working social network !

Probably the most advanced and sofisticated project I've done so far. Familygram was at first thought for my wife and her sisters, an Instagram Clone for them to exchange pictures just between them with the impression of a legit social network!

It was my first real challenge dealing with React, a lots of new concepts needed to be understood for being able to make such project. For the DataStorage part I chose to go with Firebase. Real saver for anyone not really liking dealing with Db complexities. All basics functions such as Like, Comment and Share are present !

This project is my interpretation of what a good looking website should look.

Techs Used : HTML / CSS / Javascript / React / Firebase

What does it do ?

Mobile Game Inspiration

This game was mainly inspired by popular mobile games. Simple gameplay, static scene and childly characters were picked for this project. As a first game project I wanted something useable very fast !

Simplicity

As a first experience in the gaming project, I did it very simply. As it's name tells, its a shooting game. Like most mobile games, it's not supposed to be a mindblowing gameplay, but easy and fastly distracting.

Challenging

Even through a simple gameplay, I wanted to integrate a taste of challenge. That's why at certain time, the ennemies will stop popping and let some fireball falling down the player, the goal being to avoid them obviously !

What is it ?

Simple Python Shooting Game

As many little boys, I always wanted to build my own game where I always could win, where I could design everything from A to Z. Wanted something that was mine, even when building game has always in my mind something really hard.

So it was a really fun and instructive experience to build such project. It's definitly not perfect but it's fully working and for a first shooting game using Pygame, I think it's a pretty good result !

Techs Used : Python / Pygame

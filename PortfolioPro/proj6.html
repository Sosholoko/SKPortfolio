What does it do ?

Database

One of the main feature, if not the Main feature, is database. For this app, it needs the have access to a databse to store all sorts of credentials that the user want to save. For that I used SQLachemy and Sqlite3 which is a simple solution for basic data storage without getting involved with SQL.

Security

For this kind of software that involves passmords managments, the important thing is of course 'Security'. I needed a secure solution to store the passwords, a protection in case of an attack. I chose password hashing to do this. It will register the main password and credentials as a hashed chain of characters.

What is it ?

Password Manager

PassManager is, as it's name indulges, a password manager. Many of thoses softwares already exists out there but I wanted to create mine, and to have GUI for a more comfortable use.

Everybody can easily admit that one or many of his accounts is using the same password because the need of reminding many differents passwords is not taken as serious. Solutions like mine allows anyone to store secure passwords and to protect them with only one Masterpassword !

Techs Used : Python / Tkinter / SQLACHEMY / Sqlite3
